# 📊 E-Commerce Sales Data Analysis

## Project Overview

This project analyzes an E-Commerce Sales dataset using Python to discover meaningful business insights.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Project Objectives

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Trend Analysis
- Business Insights
- Recommendations

---

## Visualizations

- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Top 10 Products
- Discount vs Profit
- Correlation Heatmap

---

## Key Insights

- Technology generated the highest sales.
- High discounts reduced profit.
- Regional performance varied.
- Seasonal trends were identified.
